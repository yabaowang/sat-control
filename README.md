# 卫星姿态控制演示系统 SAT-CTRL

交互式3D卫星姿态仿真演示，卫星绕地球飞行，通过调整偏航/俯仰/滚动角观察姿态变化和执行机构工作状态。

## 功能

- 🌍 卫星绕地球轨道飞行动画
- 🎛️ 偏航(Yaw)、俯仰(Pitch)、滚动(Roll) 三轴姿态控制
- 🔴 反作用轮状态实时显示（力矩、转速）
- 🔥 大角度机动时推力器自动点火
- 📐 XYZ体轴方向箭头标注（飞行方向、天底方向）
- 🖱️ 可拖拽旋转观察视角
- ⚡ 快捷预设：侧摆观测、前视成像、倾斜观测等

## 安装运行

确保已安装 Node.js 16+

```bash
# 1. 进入项目目录
cd sat-ctrl

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev
```

浏览器打开 http://localhost:3000 即可看到效果。

## 构建部署

```bash
# 构建生产版本
npm run build

# 预览构建结果
npm run preview
```

构建产物在 `dist/` 目录，可直接部署到任何静态服务器（Nginx、Vercel、GitHub Pages等）。

## 技术栈

- React 18
- Vite 5
- Canvas 2D（纯手写3D渲染，无外部3D库依赖）
